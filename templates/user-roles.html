User Roles and Permissions

This system defines four user roles:

  • Administrator

    Permissions:

    • Create, edit, and delete all user accounts.
    • Assign user roles.
    • Manage system settings.
    • View all system data.
    • Create invitations for all roles.

  • Auditor

    Permissions:

    • View assigned audits.
    • Perform audits and record findings.
    • Collaborate with other auditors on assigned audits.
    • View relevant audit documentation.
    • Create invitations for reviewers and auditees for assigned audits.

  • Reviewer

    Permissions:

    • View assigned documents for review.
    • Leave comments and feedback on reviewed documents.
    • Collaborate with other reviewers on assigned documents.

  • Auditee

    Permissions:

    • View assigned documents.
    • Respond to comments and feedback from reviewers.
    • Access information relevant to their role.
**Sample Users:**

- User: John Doe (Administrator)
- User: Jane Smith (Auditor)
- User: Michael Chen (Reviewer)
- User: Sarah Lee (Auditee)
